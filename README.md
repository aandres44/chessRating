# Chess Rating Live

This aims to be an open source Chess Rating website such as:

- [Mexico Chess](https://mexicochess.com/ratings/)
- [FIDE](https://ratings.fide.com/top_lists.phtml?list=open)

## Use cases

- Consult rankings across categories (rapid, classic, JR, Female, etc)
- Profile lookup with personal rating history
- Auth with accounts for personal and admin use
- Tournament publication and info
- Club sharing
- etc...

## Tech Stack

- React + TypeScript + Vite
